# Convergence Dashboard Skill

## Purpose
Validate and perfect the Lantern OS convergence loop through a cloud operator dashboard. Provides real-time convergence monitoring, orchestrator fleet management, and operator-in-the-cloud capabilities without local dependencies.

## Scope
- Convergence loop validation and monitoring
- Cloud operator dashboard access
- Orchestrator fleet management 
- Real-time convergence metrics
- No local execution requirements
- Screenshot and access validation

## Operator Dashboard Capabilities

### Cloud Operator Access
- Single unified dashboard for convergence monitoring
- Real-time convergence loop status
- Orchestrator fleet health monitoring
- Agent activity tracking
- Issue detection and resolution tracking

### Orchestrator Fleet Management
- Multi-repository fleet coordination
- HFF scan repo monitoring
- GameMaker orchestrator repo integration
- Fleet-wide convergence metrics
- Distributed agent execution

### Convergence Validation
- Real-time convergence scoring
- Issue detection and tracking
- Fix window optimization
- Convergence trend analysis
- Operator-driven convergence decisions

## Required Tools
- Web access for cloud dashboard
- GitHub API for repository monitoring
- Screenshot capture for validation
- File system access for convergence reports
- Shell execution for convergence loop runs

## Validation Requirements
- Screenshots of dashboard state
- Access validation for cloud resources
- Convergence loop execution verification
- Orchestrator fleet connectivity
- Real-time metric accuracy

## Cloud-Only Architecture
- No local execution required
- All convergence operations via cloud
- Dashboard accessible from anywhere
- Fleet-wide visibility
- Centralized operator control

## Usage Pattern
1. Access cloud dashboard via web interface
2. Review real-time convergence metrics
3. Validate convergence loop status
4. Monitor orchestrator fleet health
5. Trigger convergence operations remotely
6. Capture screenshots for validation evidence

## Success Criteria
- Dashboard accessible in cloud
- Real-time convergence metrics accurate
- Orchestrator fleet status visible
- No local dependencies required
- Operator can validate convergence remotely
- Screenshots confirm dashboard functionality
