Updated: 2026-06-06
Status: private ledger index draft

## Purpose

Create a controlled root index for the 200+ possible patentable ideas / 207-claim packet without leaking mechanisms into public portfolio materials.

## Ledger Schema

| Field | Description |
|---|---|
| Idea ID | Stable identifier, e.g. `LANTERN-IP-001`. |
| Title | Short descriptive name. |
| Domain | Memory, orchestration, provenance, policy gate, symbolic UI, hardware, services, etc. |
| Public-Safe Summary | One sentence safe to show publicly. |
| Private Mechanism | Detailed mechanism. Keep private. |
| Source Artifact | Repo path, Drive file, report title, or issue. |
| Earliest Evidence Date | Earliest known timestamp. |
| Evidence Class | repo_verified, report_verified, concept_only, patent_candidate, counsel_needed. |
| Implementation Status | not built, prototype, working, tested, shipped. |
| Novelty Signal | low, medium, high, unknown. |
| Prior Art Risk | low, medium, high, unknown. |
| Disclosure Risk | public, semi-private, private, sensitive. |
| Counsel Status | not reviewed, ready for counsel, reviewed, filed, abandoned. |
| Next Action | search, prototype, document, narrow, file, hold, retire. |
