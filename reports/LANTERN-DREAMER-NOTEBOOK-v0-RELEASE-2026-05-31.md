## Known Limitations (v0)

1. **GPT Web API**: Requires ChatGPT authentication for online mode. Offline fallback always works.
2. **Playwright browsers**: E2E tests require `npx playwright install chromium` first.
3. **Discord bot**: Requires `DISCORD_TOKEN` env var to be set.
4. **Ollama provider**: Optional local LLM at `http://127.0.0.1:11434`.
5. **Anthropic provider**: Requires `ANTHROPIC_API_KEY` env var.
