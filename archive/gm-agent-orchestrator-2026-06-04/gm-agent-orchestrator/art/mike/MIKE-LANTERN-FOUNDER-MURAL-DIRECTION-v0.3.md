# Mike Lantern Founder Mural Direction v0.3

Status: canonical correction.
Reason: prior v0.1/v0.2 output was useful as a safe placeholder, but it reads too simple, flat, and childlike for the intended Lantern OS founder-grade visual system.

## Verdict

The previous Mike Lantern Door SVG is **not** the final art direction.

It remains useful only as:

- safe placeholder geometry;
- motif map;
- renderer test fixture;
- repo proof that Canva is not required.

It should not be treated as finished key art, finished DND art, or final founder mural art.

## Correct Target

Target visual class:

```text
Lantern OS founder-grade tabletop key art / Renaissance storm-fresco mural / dark heroic systems myth
```

The image should feel like:

- a serious fantasy campaign mural;
- a founder mythology plate;
- a dark illuminated manuscript crossed with science-fiction systems art;
- a GameMaker title-screen candidate;
- an album-cover-grade Lantern OS artifact;
- adult, restrained, symbolic, and high-detail.

## Canonical Composition

Scene:

```text
A dwarf urban druid / engineer-priest in heavy draped power armor stands before a stormlit wish door. He holds a pure-light ring lantern that acts as the proof-of-work signal. Two masked guardian silhouettes flank the door, their wings made from fog, spectral math, and lantern light. Behind the door, a masked adventuring party enters a threshold of marble clouds, comet arcs, gravitational-wave bands, tesseract wireframes, and spectrograph pixels.
```

## Mood

| Axis | Requirement |
|---|---|
| Age tone | adult / founder-grade / mythic |
| Emotion | brave, restrained, solemn, operational |
| Lighting | chiaroscuro, stormlit, gold/cyan spectral light |
| Surface | fresco, marble, smoke, scratched metal, vellum |
| Detail | high density, layered symbolic geometry |
| Faces | none / masked / occluded |
| Body safety | fully armored and fully draped |

## Palette

Preferred palette:

| Color | Use |
|---|---|
| black-blue | storm field / depth |
| aged marble gray | clouds / fresco base |
| tarnished gold | lantern proof-light |
| cyan-white | wish door and spectral geometry |
| muted violet | tesseract shadow / dimensional edges |
| rust red accents | danger / comet heat, used sparingly |

Avoid candy saturation, primary-color toy palettes, and friendly mascot contrast.

## Motif Requirements

| Motif | Implementation |
|---|---|
| Lantern ring | central hard light source, not a cute halo |
| Wish door | monumental threshold, not cartoon portal |
| Tesseract | architectural/math overlay, partially embedded in clouds |
| Spectrograph pixels | signal bands, evidence artifact, should feel technical |
| COMET LEAP | high-energy launch arc, not decorative rainbow |
| Guardians | solemn silhouettes, masked and fully draped |
| Party | implied figures, no faces, entering threshold |

## Renderer Contract

A renderer may generate:

- SVG blocking layout;
- raster painterly comp;
- black-and-white ink plate;
- GameMaker title-screen crop;
- report cover crop.

A renderer must not generate:

- childlike cartoon style;
- visible real faces;
- nudity or erotic framing;
- copied official DND assets;
- Canva-dependent hidden output claims;
- wallet/revenue claims.
