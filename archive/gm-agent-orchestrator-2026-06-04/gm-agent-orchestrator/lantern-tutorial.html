STEP 4
Get Your Claude API Key (From Anthropic Website)
📋 5-Minute Process:
1. Open your web browser
2. Go to: https://console.anthropic.com/
3. Sign in with your Google account
4. Click "API Keys" on the left side
5. Click "Create Key" button
6. A long code appears (starts with sk-ant-)
7. Copy it: Ctrl+C
8. Come back to Lantern window
https://console.anthropic.com/ ↓ [Sign In with Google] ↓ Click: API Keys (left menu) ↓ Click: Create Key ↓ COPY this: sk-ant-xXxXxXxXxXx... ↓ Go back to Lantern
🔊 "Click on the link to console-dot-anthropic-dot-com. Sign in with your Google account. On the left side, you'll see a menu. Click 'API Keys.' Then click 'Create Key.' A long code appears on the screen, starting with 'S-K dash A-N-T.' Copy the entire code by selecting it and pressing Ctrl-C. Then come back to the Lantern window."

⌨️ In the Lantern Window:

3. Press Tab (moves to Save button)
4. Press Enter (saves your key)

♿ Accessibility Features Built-In

  • Keyboard Only: Navigate and control Lantern without touching a mouse
  • Screen Reader: Works with NVDA and JAWS (all buttons labeled)
  • High Contrast: White text on black (WCAG AAA standard)
  • Large Text: 14–18pt minimum font size, readable from 2 feet away
  • Clear Focus: Blue border shows exactly which button you're on
  • Simple Language: No jargon, plain English throughout
  • Visual + Text: Emojis and labels for every control
  • Easy Undo: If you make a mistake, just go back and fix it

🆘 Getting Stuck?

❌ "Cannot find the API Key field"

Press Tab over and over. You'll cycle through every button on the screen. When you see a blue border around a text field, you've found it. Type your API key there.

❌ "API Key not working"

Make sure you copied the ENTIRE key — it's very long (starts with sk-ant-). No spaces before or after. Go back to console.anthropic.com, make a NEW key, and try again.

❌ "Claude is not responding"

Check your internet connection. Try asking a simple question first: "Hello". If still stuck, open PowerShell and run:

python scripts/lantern-provider-auth.py status

This shows your configuration.

❓ "Want to add Gemini as backup?"

Follow the same steps (Step 3 again) but click the Gemini card. Get your key from https://makersuite.google.com/. Then "Set Fallback Provider" so Claude tries first, Gemini as backup if Claude is unavailable.
