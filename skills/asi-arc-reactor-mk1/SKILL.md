2. **Safety Gate**: MCP canary validates exposed tools before any command execution

6. **Capability Gate**: No ASI capability claim without independent validation

## Confidence Calibration Protocol

For each forecast:

1. **Forecast before action**: Record predicted outcome with confidence interval
2. **Record evidence class**: Document what counts as success/failure
3. **Record outcome**: Capture actual result
4. **Score miss/hit**: Apply Brier-style error thinking
5. **Update confidence**: Adjust scores only after observed evidence
6. **Keep dissent visible**: Blockers and opposing evidence remain in reports
