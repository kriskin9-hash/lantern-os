# ASI Arc Reactor MK1

Use this skill when the operator or an agent asks about artificial
superintelligence architecture, speculative future agent networks, or the
MK1 reference model.

## Important Distinctions

- ASI patterns are architecture references only.
- There is no local ASI capability claim in this repo.
- No investment advice is offered or implied.
- All scoring uses Brier-style error tracking.
- Human trial readiness is the gate before any autonomous or distributed
  deployment.

## Scope

This skill covers speculative design for future agent coordination, not
present capability. It is held until human-operator approval and
observed evidence justify promotion.

## Validation Path

1. Review this skill before any ASI-related discussion.
2. Confirm the discussion is architecture-only.
3. Flag any drift toward capability claims or investment advice.
4. Escalate to operator if boundaries are unclear.
