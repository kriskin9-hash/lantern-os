## Canonical Anchors

### Dream Journal / ORION

Dream Journal is the public product surface: a privacy-aware symbolic journaling app for dreams, notes, places, characters, lore, symbols, emotional tagging, and reflective AI interaction.

Portfolio framing: Dream Journal is the user-facing layer; Convergence IO is the engineering credential; CSF/CADD is the memory and export layer.

### Convergence IO

Convergence IO is the runtime brain: provider routing, capability claims, negative authority profiles, data classification, and provenance. It replaces monolithic agent profiles with composable primitives.

Recommended public phrase:

> Convergence IO is a composable AI orchestration layer that classifies input, checks authority, verifies capabilities, routes through provider fallbacks, and records provenance.

### CSF / CADD

CSF is the archive and retrieval layer for structured/symbolic data. CADD is the intake/distillation pattern for turning raw artifacts into docked knowledge.

Recommended public phrase:

> CSF/CADD forms a searchable knowledge archive and ingestion pipeline for structured, symbolic, and evidence-labeled project memory.

### Synthesasia

Synthesasia is the invented symbolic-emotive language. Canonical update: Synthesasia uses three symbols or three-symbol groupings to emote.

Initial rule:

> A Synthesasia utterance is a triad: symbol 1 sets the emotional origin, symbol 2 sets the motion or tension, and symbol 3 sets the landing / response.

### Tesseract / 3^12

The tesseract layer is a symbolic and computational architecture. It should be described as a model for layered memory, observer focus, and sparse active state, not as a literal physics claim.

Safe phrase:

> The Lantern tesseract is a symbolic architecture for sparse state navigation: a large latent address space with only a small active wavefront rendered at a time.

### Private IP / Invention Backlog

The internal invention backlog is large and should be treated as private. Public pages should say "evidence-indexed invention backlog" rather than listing sensitive claim mechanics.
