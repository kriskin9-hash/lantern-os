# Lantern OS Receptionist Call List

Status: operator review sheet
Date: 2026-05-28

## Purpose

This sheet gives a receptionist a safe call path for Lantern OS outreach. It uses organization switchboards and public program contacts only. Do not add personal phone numbers, private mobile numbers, scraped direct dials, or unverified contacts.

## Call Rules

- Call only public organization numbers or numbers the operator already owns.
- Ask for the correct department before pitching.
- Record the receptionist name only if they freely provide it as part of the call.
- Do not imply funding, partnership, endorsement, medical benefit, government approval, or school approval unless it is already documented.
- Do not request children’s personal information.
- Do not send attachments until the recipient asks for them or provides the proper intake address.
- If the organization asks for a web page, send the public repo or support page only.

## Boundary

This list is a routing sheet, not proof of endorsement. All numbers should be rechecked against the organization’s official website before a calling campaign. If a number changes or a receptionist asks not to be called, mark it held and use the official web intake path instead.
