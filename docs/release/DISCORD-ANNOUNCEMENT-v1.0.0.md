Dream Journal v1.0.0 just shipped.

Here is what landed: voice capture (tap the mic, speak your dream), interactive search by text or tag, CSV and JSONL export, and 8 entry kinds — Dream, Note, Place, Character, Event, Lore, Symbol, Mirror. Everything runs locally. No cloud, no account, no data leaving your machine.

To try it: git pull origin master, then npm start --prefix apps/lantern-garage, and open http://127.0.0.1:4177.

First thing to try: hit the mic button right after you wake up and just talk. Tag it as a Dream. Done. Your first entry is in.

The 3 Door Game — the interactive symbolic layer — is designed and documented, and ships fully in v1.0.1. This is the foundation. It is solid.

Go capture something. 🌙
